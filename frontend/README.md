# FinSage Frontend

React frontend for the FinSage personal finance management application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

Frontend runs on `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Technologies

- React 18
- Tailwind CSS
- Vite
- React Router
- Recharts
- Axios
